# SKILLS.md

## Agent Skills & Capabilities Matrix

### Current AI Tools & Skills
- **VSCode + Continue.dev**: Full repository navigation, code editing, spec following for the ToDoList project.
- **OpenSpec Agent**: Spec-driven workflow execution, artifact management, reference-based governance.
- **General Capabilities**: Modern web development, documentation, YAML/Markdown, Git workflows, ToDoList domain logic.

### Required Skills for Agents
- Understand reference-based ownership model (`openspec/config.yaml`).
- Maintain zero unnecessary duplication.
- Work with both industry-standard root files and OpenSpec extensions.
- Support AI-vendor-agnostic practices.

### Extension Point
Tool-specific or advanced skills are defined in `openspec/specs/skills-extension.spec.md` (if created).
